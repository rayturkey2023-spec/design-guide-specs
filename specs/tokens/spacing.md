# 间距 Token

> 所有内外边距必须从以下 Token 取值，禁止使用 Token 以外的任意间距值。

## 间距值

| Token | 值 | 典型使用场景 |
|-------|-----|------------|
| spacing-2 | 【填入 px】 | 图标与文字间距、极小内间距 |
| spacing-4 | 【填入 px】 | 标签内间距、图标对齐微调 |
| spacing-8 | 【填入 px】 | 组件内元素间距、行内元素间距 |
| spacing-12 | 【填入 px】 | 卡片内子元素间距 |
| spacing-16 | 【填入 px】 | 卡片内边距、页面左右边距（标准）|
| spacing-20 | 【填入 px】 | 较大内边距、区块间距 |
| spacing-24 | 【填入 px】 | 模块间距、列表项垂直间距 |
| spacing-32 | 【填入 px】 | 大模块间距、页面顶部留白 |

## 间距使用约束

- 禁止使用 Token 以外的值（如 10px、15px、22px）
- 相邻模块间距不小于 spacing-16
- 页面左右安全边距统一为 spacing-16（移动端）
