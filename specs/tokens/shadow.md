# 阴影 Token

| Token | 值（CSS box-shadow 格式）| 使用场景 |
|-------|------------------------|---------|
| shadow-sm | 【填入，如 0 1px 4px rgba(0,0,0,0.08)】 | 卡片轻浮层效果 |
| shadow-md | 【填入，如 0 4px 12px rgba(0,0,0,0.12)】 | 下拉菜单、浮动操作按钮 |
| shadow-lg | 【填入，如 0 8px 24px rgba(0,0,0,0.16)】 | 弹窗、底部半浮层 |

## 阴影使用约束

- 同一页面阴影层级不超过 2 种，避免视觉混乱
- 页面背景上的卡片使用 shadow-sm，浮层类组件使用 shadow-lg
- 禁止对文字或图标直接添加阴影
