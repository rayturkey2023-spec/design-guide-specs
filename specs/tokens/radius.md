# 圆角 Token

| Token | 值 | 使用场景 |
|-------|-----|---------|
| radius-sm | 【填入 px】 | 标签、Badge、小型元素 |
| radius-md | 【填入 px】 | 按钮、输入框 |
| radius-lg | 【填入 px】 | 卡片、图片容器、底部浮层顶部圆角 |
| radius-full | 999px | 胶囊按钮、头像、圆形图标 |

## 圆角使用约束

- 同一组件内不混用不同圆角 Token
- 底部浮层：仅顶部两角使用 radius-lg，底部无圆角
- 卡片内嵌图片：图片圆角与卡片圆角保持一致或略小
