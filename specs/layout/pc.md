# PC 端布局规范

## 基础页面结构

```
屏幕
├── 顶部导航（全宽）
│   ├── 高度：【填入 px，建议 60px】
│   ├── 背景：color-bg-card
│   └── 内容最大宽度居中对齐
├── 内容区
│   ├── 最大宽度：【填入 px，建议 1200px】，水平居中
│   ├── 左右边距：spacing-24 或 spacing-32
│   └── 顶部间距：spacing-24
└── 底部 Footer（如有）
```

## 栅格系统

- 列数：【填入，建议 12 列】
- 列间距（gutter）：【填入 px，建议 24px】
- 常用布局比例：
  - 主内容区：8 列 / 辅助区：4 列
  - 全宽内容：12 列
  - 卡片列表：每行 3 列（4 列/个）或 4 列（3 列/个）

## 触控与交互规范

- hover 态：背景色变为 color-bg-hover，过渡时长 150ms
- 操作按钮尺寸：Medium（比移动端宽松）
- 表格行高：【填入 px，建议 48px】
